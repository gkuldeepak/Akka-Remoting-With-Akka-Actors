# Akka-Remoting-With-Akka-Actors
This template is about setting up and running basic akka remote with akka actor

## This template uses - 
- Akka Actors
- Akka Remote

## Pre-Requisites - 
- Akka
- Scala
- Java

## Getting Started - 
- Use `sbt clean compile` to compile the program.
- Run the program by `sbt run`.
